# ADR 0006: Startkategorien fuer Komponenten und Zusatzartikel

## Status

accepted

## Kontext

Das Komponentenmodell soll erweiterbar sein, aber zum Start nicht unnoetig viele Kategorien enthalten.

## Entscheidung

Zum Start werden folgende Kategorien vorgesehen:

- iPad
- Pencil
- Tastatur
- Adapter
- HDMI-Kabel
- Netzteil
- Kabel
- Magic-Maus

Weitere Kategorien werden erst bei Bedarf ergaenzt.

## Konsequenzen

- Das Startmodell bleibt ueberschaubar.
- Neue Kategorien koennen spaeter ohne Schemaaenderung angelegt werden.
- Import und UI koennen sich zunaechst auf die bekannten Kategorien konzentrieren.
